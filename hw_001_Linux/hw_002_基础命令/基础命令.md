## 013.判断命令为内部还是外部

`type 命名名`

> 如果告诉了命令的地址,则为外部命令
>
> 内部命令提示 `xxx is a shell builtin`

- 原理: 根据$PATH变量来寻找的
- 内部命令
  - 直接在内存中
- 外部命令
  - 找到以后加载到内存中

## 025.屏幕共享

`yum -y install screen` 安装命令

`rpm -ivh xxx.rpm` 安装rpm包的命令(参考)

步骤:

- 1.开启屏幕共享 `screen -s help`
- 2.用另一个终端连接到同一台服务器
- 3.查看有哪些屏幕共享 `screen -ls`
- 4.连接屏幕共享 `screen -x help`
- 5.退出 `exit`
